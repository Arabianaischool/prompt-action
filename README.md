# Prompt Action Skill

A Claude skill that automatically transforms simple user requests into professional prompts and executes them directly in the same conversation.

## Why this skill exists

Many people say that you must write a professional prompt to get the best results from AI.

But usually one of two things happens:

You either feel too lazy to write all the details

or you simply do not know how to write a strong prompt.

So people go to external prompt improvement tools, which wastes time and consumes extra tokens.

This skill removes that problem.

## How it works

When the skill is enabled you can write something very simple like:

اعمل لعبة XO

The skill will automatically:

1. Convert your request into a professional prompt
2. Treat it as if written by an expert in the required domain
3. Execute the improved prompt
4. Return the final result in the same conversation

Example:

User request

Make a XO game for me

The skill converts it into a professional prompt for building an interactive game and Claude generates the playable result immediately.

## Benefits

Faster workflow

Less effort writing prompts

Better AI responses

Lower token usage
